时间线

星际争霸时间线覆盖了自16世纪到2501年三族历史的关键时刻（2501：吉姆雷诺在母巢之战4年后返回Mar Sara 玛萨塔行星）本时间线关键事件来自星际1游戏剧情、星际争霸官方系列小说、星际争霸官方漫画及其他官方授权纸质媒体注：本文所有译名来自星际1官方剧情和星际2剧情官方中文介绍。

  • 1500

    一群被母星Aiur（艾尔）放逐的protoss流亡者拒绝通过加入Khala。这些流亡者称自己为黑暗圣堂，定居于行星Shakuras（萨库拉斯）。这次分离行为导致Protoss种族分裂为两个派系，史称 the Discord。

  • 1865

    　黑暗圣堂武士Zeratul（泽拉图）诞生。若干年后他将帮助分裂的Protoss社会走向统一。

  • 2143

    Tassadar（塔萨达）诞生。后来他将成为Protoss母星Aiur的执政官。

  • 2259

    四艘超级航母 — Argo, Sarengo, Reagan, Nagglfar从地球运输罪犯时，在克普鲁星区迫降。幸存者开荒发展了行星莫里亚,乌莫加,和塔尔苏尼斯，以此建立了一个新的人类社会，并向其他行星扩张。

  • 2333

    通过在其他行星建立殖民地，Tarsonis（塔尔苏尼斯）成为了Terran Confederacy（人类联邦）的首都 —— 这是一个强大的专制政府。

  • 2460

    　Arcturus Mengsk（阿克图斯·蒙斯克）诞生。他成为了联邦精英的一员。

  • 2464

    Tychus Findlay （提卡斯·芬得利）诞生。他将在Guild Wars（行会战争）里成为Jim Raynor（吉姆雷诺）的好朋友。

  • 2470

    Jim Raynor（吉姆雷诺）诞生于Shiloh（夏伊洛）边远之地，Trace（崔丝）和Karol Rayno（卡罗尔雷诺）之家。

  • 2473

    Sarah Kerrigan（莎拉·卡瑞根）诞生。她天生具有强大的PSI灵能。

  • 2478

    Arcturus Mengsk（阿克图斯·蒙斯克）从Styrling Academy（斯蒂尔灵学院）毕业，并不顾父母的意愿报名参军，成为联邦海军陆战队的一员。

  • 2485

    作为对联邦卑劣进行资源掠夺的还击，Morian Mining Coalition（莫里安矿工联盟）和 Kelanis 运输行会联合发动了武装起义，建立了柯尔-莫里安联盟。战争爆发,这次战争史称Guild Wars（行会战争）。

  • 2488-2489

    吉姆雷诺加入联邦海军陆战队，并在此碰到了Tychus Findlay。之后联邦政府同柯莫联盟之间的战争，第321殖民地突击营（成员包括雷诺和芬得利）首当其冲，一战成名，被称为天堂的恶魔。

  • 2488-2489

    吉姆雷诺在柯莫联盟的监狱遇见了联邦士兵Cole Hickson（柯尔希克森）。这段时间内，希克森教授雷诺如何应付柯莫联盟的残酷拷问存活下来。

  • 2488-2489

    在行会战争结束前，吉姆雷诺和提卡斯·芬得利决定自行脱离联盟政府军队。与此同时阿克图斯 蒙斯克在获得上校军衔后从联盟军队退役。之后他成为了银河系边缘一名成功的勘探者。

  • 2488-2489

    在行会战争四年后，联邦政府通过和柯莫联盟的“和平谈判”，分化了柯莫联盟几乎所有的矿工行会支持者。尽管受到如此大的挫折，柯莫联盟仍获允继续存在，并得到了自治权。

  • 2488-2489

    阿克图斯·蒙斯克的父亲是联邦议员安格斯·蒙斯克。联邦政府派遣了三名幽灵特工刺杀了安格斯和他的妻女。阿克图斯悲愤欲绝，他接过了柯哈反抗力量的领导权，并掀起了反抗联邦的游击战。

  • 2491

    联邦政府对柯哈四进行了核打击，数百万人死亡。作为报复，阿克图斯 蒙斯克加强了对联邦的武装反抗。这段时期，阿克图斯解放了一名联邦幽灵特工——莎拉 卡瑞根，之后她成为了蒙斯克的副指挥官。

  • 2495

    提卡斯·芬得利被当局逮捕，吉姆雷诺结束了犯罪生涯，在玛萨拉行星开始了联邦治安官的新生活。

  • 2499 - 2500

    克普鲁星区出现了两类异形威胁：残酷无情，拥有极强适应能力的Zerg族和神秘莫测的Protoss族相继出现于此。Protoss对人族行星Chau Sara无端发动攻击，联邦政府非常愤怒。

  • 2499 - 2500

    而大部分人类不知道，Chau Sara行星已经完全被Zerg所感染侵占，Protoss的攻击在于毁灭这些入侵者。另一方面，在附近的玛萨拉行星也发现了Zerg的活动迹象。

  • 2499 - 2500

    在玛萨拉行星，吉姆雷诺摧毁了Backwater Station(死水站),一个被Zerg感染的人类前哨站，并因此被联邦监禁。不过很快蒙斯克的反抗部队柯哈之子来到此地，吉姆雷诺获得释放。

  • 2499 - 2500

    一名联邦陆战队员Ardo Melnikov（安多·梅尔尼科夫）发现自己陷入了玛萨拉的纷争。他承受着行星Bountiful的前世痛苦回忆，但很快，他发现关于他的过去真相更加黑暗。

  • 2499 - 2500

    玛萨拉和Chau Sara遭受了同一命运，被Protoss毁灭，吉姆雷诺、阿克图斯蒙斯克、柯哈之子以及部分行星居民逃出生天。

  • 2499 - 2500

    吉姆雷诺感到被联邦所背叛，于是他加入了柯哈之子，并在此与萨拉凯瑞根相识。一名环球新闻网（UNN）的记者，Michael Liberty(迈克尔利伯蒂)伴随反抗力量一路报道，反击联邦的新闻宣传。

  • 2499 - 2500

    一名联邦政客Tamsen Cauley派遣War Pigs（好战者）部队 —— 这是一支专为联邦处理各种肮脏工作的秘密部队——刺杀阿克图斯蒙斯克。不过刺杀行动失败。

  • 2499 - 2500

    Tarsonis（塔尔苏尼斯）的联邦精英的女儿，在她感应到父母兄弟被杀后释放了自己的潜在PSI灵能。在得知其恐怖能力之后，联邦政府将其抓获，并试图利用她的天赋能力。

  • 2499 - 2500

    阿克图斯蒙斯克在联邦首都塔尔苏尼斯部署了毁灭性武器——PSI发射器。这个装置将会放大发送PSI灵能信号，引来大批Zerg。首都塔尔苏尼斯就此沦陷给了联邦政府致命的打击。

  • 2499 - 2500

    阿克图斯蒙斯克背叛了莎拉凯瑞根，将其遗弃在塔尔苏尼斯的虫群之中。愤怒的吉姆雷诺退出柯哈之子，并组织了自己的反抗部队Raynor's Raiders（雷诺突击队）。

  • 2499 - 2500

    很快他发现了卡瑞根的真正命运：卡瑞根并未被Zerg所杀，而是被其变异成为一个新的强大存在——刀锋女皇。

  • 2499 - 2500

    阿克图斯蒙斯克宣布加冕为皇帝，控制众多人类行星的帝国在克普鲁星区成立。

  • 2499 - 2500

    帝国参议院 Corbin Phash 发现他的幼子 Colin（科林）拥有吸引致命Zerg军队的PSI灵能，帝国认为这是一件可以利用的武器。

  • 2499 - 2500

    在边缘之地Bhekar Ro，terran，protoss和zerg三方部队为新近发掘出土 Xel'Naga遗迹归属而展开战斗。萨尔纳加被认为是一个古老的外形种族，可能影响过Zerg和Protoss的进化过程。

  • 2499 - 2500

    Zerg最高统治者主宰发现了Protoss母星Aiur艾尔所在，Zerg大批入侵。但英勇的高阶圣堂武士Tassadar（塔萨达）牺牲了自己毁灭了主宰。

  • 2499 - 2500

    然而大部分Aiur已经毁灭，剩余Protoss舰队通过萨尔纳加传送门来到了黑暗圣堂的萨库拉斯行星。这是黑暗圣堂被从Aiur流放后双方第一次接触。两个神族社会重新统一。

  • 2499 - 2500

    Zerg通过传送门追赶Protoss难民来到萨库拉斯。吉姆雷诺同塔萨达和黑暗圣堂泽拉图结盟，留在Aiur负责关闭传送门。

  • 2499 - 2500

    与此同时泽拉图和Protoss执政官Artanis(阿塔尼斯)利用远古萨尔纳加神庙的力量净化了被Zerg完全侵袭的行星。

  • 2499 - 2500

    United Earth Directorate (UED)（地球联合理事会）观察到了三族间的战斗，他们从地球来到克普鲁星区，意图接管统治。为了达到此目标，UED在Zerg占领的Char(查尔)行星捕获了新生的主宰幼虫。

  • 2499 - 2500

    刀锋女皇、蒙斯克、雷诺以及Protoss此时抛开了彼此的纷争，试图合作击败UED和新生的主宰。这个不可能的联盟杀死了第二主宰，刀锋女皇目标实现，接管了克普鲁所有的Zerg。

  • 2499 - 2500

    在查尔的一颗未知卫星上，泽拉图遭遇了人类 Samir Duran（萨米尔·杜安），刀锋女皇的前盟友。他发现杜安已经成功将Zerg和Protoss的DNS混合，创造了一个新物种。

  • 2499 - 2500

    阿克图斯蒙斯克消灭了半数的幽灵特工，为确保他们忠心不二，他接管了前联邦政府的幽灵特工计划，并将其融入帝国的幽灵项目。此外，他在行星柯哈四的一个卫星Ursa上建立一个新的幽灵学院。

  • 2499 - 2500

    年轻的科林被帝国捕获，并被送到了幽灵学院，与此同时，他的父亲科本在乌莫加保护国发表了反对帝国的言论。作为回应，科本成为了帝国的暗杀对象。

  • 2501

    诺娃 特拉从被毁灭的家园塔尔苏尼斯逃了出来，在幽灵学院受到另一个PSI天赋者的训练，锻炼她的PSI灵能 。

  • 2502

    阿克图斯 蒙斯克的儿子瓦里安出生，阿克图斯意图让瓦里安继承蒙斯克王朝阿克图斯召回他的儿子瓦里安，打算让其成为皇帝，而瓦里安则对此十分冷漠。

  • 2502

    球新闻网的记者 Kate Lockwell（凯特·罗克韦尔）进入帝国部队，负责进行爱国宣传。在这段服役期，她遇见了前环球新闻网记者迈克尔利伯蒂，并发现了隐藏于帝国表面之下的黑暗真相。

  • 2502

    Tamsen Cauley 为了掩该其刺杀阿克图斯蒙斯克的行动，计划消灭好战者部队成员——该部队已经解散。在行动前，他先召集了好战者去杀死吉姆雷诺，他认为这次行动可以获得蒙斯克的欢心。

  • 2502

    而负责这次行动的成员则是前联邦士兵柯尔 希克森，他在莫里安监狱从柯莫联盟的拷问里拯救雷诺。

  • 2502

    来自克普鲁星区的三族军队为了控制行星Artika（阿提卡）上的远古萨尔纳加神庙而陷入混战。其间，一些好战分子为了个人目标也参与了这场混战。

  • 2502

    柯莫联盟工作人员来到了一个荒凉的行星，希望能发掘一些有价值的遗迹。在发掘过程中，他们发现了该行星居民失踪的可怕真相。

  • 2502

    一个神经质的维京飞行员Jon Dyre上尉在一次武装练习中攻击了Ursa的无辜殖民者。Wes Carter（韦斯卡特）将面对他的前老师，并终结其疯狂的杀戮行为 。

  • 2502

    Sandin Forst（山迪福斯特），一位熟练的雷神驾驶员，同他的两名忠实队友一起勇敢的进入了玛萨拉的一处人族废墟，在取得任务物品后，福斯特发现了一个以前从未被发现的秘密。

  • 2503

    帝国科学家捕获Protoss执政官Muadun，对其进行实验以获取Protoss PSI灵能完全形态的信息——the Khala。由Stanley Burgess博士带领，研究人员使用各种违反伦理道德的手段来研究PSI灵能。

  • 2503

    考古学家Jake Ramsey（杰克·拉姆西）对一座萨尔纳加神庙进行了研究， 但情况很快失去了控制，杰克受到了一个神秘的Protoss保护者影响，他的思维与之融合，无数Protoss历史记忆涌现出来。

  • 2503

    杰克 拉姆西继续来到艾尔行星进行探险，在脑海中的Protoss保护者指令下，他发现了行星地下的暗影迷宫，发现了一个或许可以拯救宇宙的神圣水晶。

  • 2503

    一些受过神秘训练的帝国幽灵特工开始失踪。诺娃·特拉从幽灵学院毕业，开始调查失踪事件，并发现了一个恐怖的秘密。

  • 2503

    杰克·拉姆西在穿越艾尔的萨尔纳加传送门时同保镖Rosemary Dahl分开，Rosemary被杀死。而杰克在时间耗尽之前将Protoss保护者从自己的思维里分离出去，否则两者都将死亡。

  • 2503

    Moebius Foundation（一个对外星遗物感兴趣的神秘人类组织）的一个小队调查了一座克普鲁星区的萨尔纳加建筑。在研究中，这些科学家发现了一股潜伏在古代遗迹中的黑暗力量。

  • 2503

    Kern 从帝国的死神部队退役后（reaper台服译作死神，个人觉得从原画和习惯译法来说，死神更有韵味）打算开始自己的新生活。但是他难以摆脱过去的麻烦，一名前队友不期而至。

  • 2503

    一名夜总会歌手Starry Lace 发现自己正处于帝国和柯莫联盟外交纷争的中心。

  • 2504

    遁世的吉姆雷诺重回玛塔萨。
